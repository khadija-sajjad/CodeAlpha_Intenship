# Task 1: Web Scraping

## Objective
Scrape book data from books.toscrape.com using Python and BeautifulSoup.

## Tools & Libraries Used
- Python 3.13
- BeautifulSoup4
- Requests
- Pandas
## What I Did
- Scraped 1000 books data from 50 pages
- Extracted Title, Price, and Rating of each book
- Saved data in CSV format
## Dataset
- Total Records: 1000 books
- Columns: Title, Price, Rating
## Files
- `web_scraping_books.py` - Python scraping script
- `books_data.csv` - Scraped dataset
