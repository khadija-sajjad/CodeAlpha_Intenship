# Task 3: Data Visualization

## Objective
Transform books dataset into meaningful visual insights using Python visualization libraries.

## Tools & Libraries Used
- Python 3.13
- Matplotlib
- Seaborn
- Pandas

## What I Did
- Created a professional dark-themed dashboard
- Visualized price distribution with mean line
- Analyzed rating distribution across 1000 books
- Explored price vs rating relationship
- Compared price ranges across all rating categories

## Charts Created
- Price Distribution Histogram
-  Rating Distribution Bar Chart
- Price vs Rating Scatter Plot
- Price Range by Rating Box Plot

## Key Insights
- Average book price: £35.07
- Most common rating: One star
- Price range: £10.00 - £59.99
- No significant correlation between price and rating
